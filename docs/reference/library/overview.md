# Overview

While The Harmony Language Reference describes the exact syntax and semantics of the Harmony language, this library reference manual describes the standard library that is distributed with Harmony. These modules provide additional functionality and common structures that are often used in concurrent programming.

 - [alloc](./alloc.md)
 - [bag](./bag.md)
 - [hoare](./hoare.md)
 - [list](./list.md)
 - [set](./set.md)
